# Preferences

## Change UI Appearance

The default dark appearance helps you stay focused on the artworks you are working on. However, you can change the UI appearance to light in the theme setting:

1. Choose **Menu > Theme**
2. Click either Dark or Light.

## Change UI Language

1. Choose **Menu > Language**
2. Choose a preferred language.

## Change Units

If you are creating vector graphics for printing, you would need physical units like millimeters, centimeters or inches. To choose the preferred unit for measurement:

1. On the right side of the menubar, click on the units field <InlineIcon d={icons.rule} />.
2. Choose a preferred unit from the popup menu.
