# Keyboard Shortcuts

## File Commands

| Shortcuts | Windows                   | macOS                          |
| --------- | ------------------------- | ------------------------------ |
| Undo      | <kbd>ctrl + z</kbd>       | <kbd>command + z</kbd>         |
| Redo      | <kbd>shift ctrl + z</kbd> | <kbd>shift + command + z</kbd> |
| Cut       | <kbd>ctrl + x</kbd>       | <kbd>command + x</kbd>         |
| Copy      | <kbd>ctrl + c</kbd>       | <kbd>command + c</kbd>         |
| Paste     | <kbd>ctrl + y</kbd>       | <kbd>command + y</kbd>         |
