# Frame

## Draw Frames

<Steps>

### Select the frame tool

Click the frame tool <InlineIcon d={icons.frame}/> in the toolbar.

### Create a frame

Do one of the following:

- Click and drag a rectangle to create a frame.
- In the frame panel, select a preset size and click "create a frame";

</Steps>

## Rename Frames

There two ways to rename a frame:

1. In the layers panel, double click on the frame name.
2. Double click on the frame name at the top left corner of a frame.

Type a new name, then press enter or click away to change the name.

## Transform Frames

<Steps>

### Select a frame

Select the selection tool <InlineIcon d={icons.selection}/> in the toolbar, then click on the frame name at the top left corner of a frame. Press <kbd>shift</kbd> to select more frames.

### Resize or Drag the frames

Do one of the following:

- Drag on a frame name to move the selected frames to a new position.
- In the frame panel, select a new size in the presets to resize the frames.
- In the frame panel, click the orientation button <InlineIcon d={icons.orientation}/> to change the orientation to landscape or portrait.
- Drag one of corners or sides of the bounding box to resize all selected frames.

</Steps>
