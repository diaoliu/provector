# Image

## Insert Images

1. Click the image tool <InlineIcon d={mdiImageOutline} /> in the toolbar.
2. Move your cursor to a point where you want to place the images, and click on the canvas.
3. Select one or more images to insert them into current project.

## Paste Images

### Paste an image from browser directly

1. Right-click on the image and select "Copy Image" from the context menu.
2. Press <kbd>ctrl/command + v</kbd> to paste it into current project.

### Paste images from your computer

1. Select one or more images in File Explorer (Windows) or Finder (Mac) and press <kbd>ctrl/command + c</kbd>.
2. Back to the app and press <kbd>ctrl/command + v</kbd> to paste the images into current project.

## Drag and Drop Images

1. Click and hold the images you want to insert from file explorer or browser.
2. Drag the images and hover over the canvas where you want to drop the images.
3. Release the mouse button to drop the images into current project.

<Callout>
  You can drop an image directly on a path or clipping mask to clip the image
  within a shape.
</Callout>
