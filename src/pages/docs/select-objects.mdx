# Select Objects

## Select Objects with the Selection Tool

<Steps>

### Active the Selection Tool

Click the Selection Tool <InlineIcon d={icons.selection} /> from the toolbar or press <kbd>v</kbd>

### Select one or more Objects

- **Select Single Object**: click on the object.
- **Select Multiple Objects**: click and drag a selection marquee (bounding box) around the objects.

### Add More Objects to the Selection

Hold down the <kbd>shift</kbd> while clicking on the object or dragging a rectangular selection.

### Deselect All Objects

Click on an empty area of the canvas.

</Steps>

## Select Objects with the Layers panel

1. Click the layers button in the top of the right sidebar to open the layers panel.
2. Click the triangle to collapse the group or frame.
3. Click an object to select it from the layers.

## Select Anchor Points with the Direct Selection Tool

<Steps>

### Active the Direct Selection Tool

Click the Direct Selection Tool <InlineIcon d={icons.directSelection} /> from the toolbar or press <kbd>a</kbd>

### Select one or more Anchor Points

- **Select Single Point**: click on the anchor point.
- **Select Multiple Points**: click and drag a selection marquee (bounding box) around the points.

### Add More Anchor Points to the Selection

Hold down the <kbd>shift</kbd> while clicking on the points or dragging a rectangular selection.

### Deselect All Anchor Points

Click on an empty area of the canvas.

</Steps>

## Select Objects within a Group or Clipping Mask

Double click a group to enter the group and select individual objects within it.

## Select All

Do one following to select all objects in the document:

- <kbd>ctrl/command + a</kbd>
- Select **Menu > Select > All**.

## Select Same

1. Select one object with the desired appearance attribute.
2. Select **Menu > Select > Same**, then select one of following:
   - Blending Mode
   - Opacity
   - Fill Color
   - Stroke Color
   - Stroke Weight
   - Font

## Select Type

Select **Menu > Select > Object**, then select one of following:

- Stray Points
- Clipping Masks
- Text Objects
